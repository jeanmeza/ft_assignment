# FSTT Assignment | Functional testing

The goal of the assignment is to **explore the use of Large language Models (LLMs) to automatically generate assertions
in Selenium WebDriver test scripts** where they are missing.

The model to used for the experiment is **Gemma-4-31B-IT**, an instruction model from Google DeepMind's Gemma family.

The tests require the [Spring PetClinic](https://github.com/spring-projects/spring-petclinic) application to be running
locally.
